INSTALA LOS COMANDOS UNO POR UNO.
TERMINALES COMPATIBLES:
• Termux
• CloudShell 
• Servidor
• Replit
• Heroku
• PC


1 ~ termux-setup-storage
( En caso se usar termux, incluye esta opción. )

2 ~ apt update && apt upgrade && pkg install -y git nodejs ffmpeg imagemagick yarn

3 ~ git clone https://github.com/NameBot/NameBot && cd NameBot
( En caso de guiar un directorio, use: cd storage/downloads/NameBot/NameBot )

4 ~ yarn install

5 ~ npm install

6 ~ npm start


NOTA:
- Recuerde que si tiene dudas o como instalarlo, puede contactar con el asistente informatico en la comunidad o al mismo ceo.
